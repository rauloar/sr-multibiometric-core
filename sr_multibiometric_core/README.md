# SR Multibiometric

Multi-brand biometric integration framework for Odoo.

## Features

- Multi-device architecture
- Driver factory
- ZKTeco support
- Fake driver for testing
- Future Hikvision support
- Attendance processing
- Cron sync
- Flash drive import ready

## Requirements

- hr
- hr_attendance

## Installation

Install `sr_multibiometric_core` first.
Then install a driver module.
