SR Multibiometric for Odoo 19

License: LGPL-3


DESCRIPTION

SR Multibiometric is a biometric integration suite for Odoo 19 Community.

The Core module provides the interface and integration with hr_attendance.

Device drivers are distributed separately.

INSTALLATION

1) Copy modules into custom-addons

2) Update Apps list

3) Install:

SR Multibiometric Core


DRIVER INSTALLATION

Drivers are not included in the core package.

Drivers must be copied manually to addons path.

Example drivers:

sr_multibiometric_zkteco
sr_multibiometric_hikvision
