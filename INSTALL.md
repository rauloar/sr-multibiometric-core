# SR Multibiometric Installation Guide

This guide describes the standard Odoo 19 module installation procedure.

1. **Copy Module**: Download `sr_multibiometric_core` and place it in your standard Odoo `addons_path` directory (e.g. `custom-addons`).
2. **Restart Odoo**: Restart your Odoo service so it scans the new directories.
3. **Developer Mode**: Log in as Administrator and activate Developer Mode from the Settings.
4. **Update Apps List**: Navigate to the native **Apps** module and click **Update Apps List** in the top menu.
5. **Install**: Search for `SR Multibiometric Core` and click on the **Install** button.

*Note: No custom installation scripts are supported or required. This module follows official Odoo deployment standards.*
